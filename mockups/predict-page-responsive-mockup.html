📱 Responsive Design Notes

  • Mobile nav (hamburger menu): On screens below 640px (sm breakpoint), nav links are replaced by a hamburger icon that reveals a slide-down menu. Click the ☰ icon above to see it in action.
  • Filter tabs: Use flex-wrap so they wrap to a second row on narrow screens rather than overflowing.
  • Match cards: Team name width shrinks from w-[120px] to w-[100px] on mobile via sm:w-[120px]. Combined with truncate, long names are handled gracefully.
  • Venue/time info: Full details shown on desktop; shortened version on mobile using hidden sm:inline / sm:hidden.
  • Sticky save bar: Works perfectly on mobile — always accessible at the bottom of the viewport. Status text shortens on small screens.
  • No major restructuring needed. The existing card-based, single-column layout already scales well. The only real addition is the mobile nav drawer.

🖥️ How to test:

  • Resize this browser window to see the responsive breakpoints in action
  • Or use browser DevTools → Device toolbar (Cmd+Shift+M) to simulate phones
  • Try iPhone SE (375px), iPhone 14 (390px), and iPad (768px)
