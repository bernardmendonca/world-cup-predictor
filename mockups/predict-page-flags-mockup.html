Design Notes

  • Flag placement: Home team flag appears to the RIGHT of the name (flag faces inward toward scores). Away team flag appears to the LEFT of the name (flag faces inward).
  • Flag size: 20×14px for match cards, 16×11px for penalty winner buttons.
  • Spacing: Team name containers widened from w-[100px]/w-[120px] to w-[120px]/w-[140px] to accommodate flags.
  • TBD teams: No flag shown when teams aren't confirmed (knockout slot labels only).
  • Team selection: Flags appear inline in both the dropdown options and the locked read-only display.
  • Penalty winner buttons: Smaller flag beside the team code in penalty selection.
  • Dark mode: Flags are unaffected by dark mode (they're images). Rounded corners soften the edges.
  • Implementation: Use a FIFA code → ISO 3166-1 alpha-2 mapping utility, then load flags from a CDN like flagcdn.com (as used in this mockup) or bundle SVG flag icons.
